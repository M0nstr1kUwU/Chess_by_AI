
# 3D Шахматы на Kool Engine (Kotlin)

## Структура проекта

```
kotlin/lesson28/
├── build.gradle              # Конфигурация сборки Gradle
├── settings.gradle           # Настройки проекта
└── src/main/
    ├── kotlin/chess3d/
    │   ├── Main.kt           # Точка входа в приложение
    │   ├── ChessLogic.kt     # Логика шахмат (правила, ходы)
    │   ├── ChessBot.kt       # ИИ бот с алгоритмом минимакс
    │   └── ChessScene.kt     # 3D сцена и рендеринг
    └── resources/
        ├── models/           # Папка для 3D моделей (.gltf)
        │   ├── white_king.gltf
        │   ├── white_queen.gltf
        │   ├── white_rook.gltf
        │   ├── white_bishop.gltf
        │   ├── white_knight.gltf
        │   ├── white_pawn.gltf
        │   ├── black_king.gltf
        │   ├── black_queen.gltf
        │   ├── black_rook.gltf
        │   ├── black_bishop.gltf
        │   ├── black_knight.gltf
        │   └── black_pawn.gltf
        └── textures/         # Текстуры для материалов
```

## Используемые библиотеки

1. **Kool Engine** (`de.fabmax.kool:kool-core`, `kool-physics`) - 3D движок для рендеринга
2. **ChessLib** (`com.github.bhlangonijr:chesslib`) - Библиотека для логики шахмат и AI
3. **Kotlin Logging** - Логгирование событий
4. **SLF4J** - Фасад для логгирования

## Как добавить свои 3D модели

### Требования к моделям:
- Формат: `.gltf` или `.glb` (binary glTF)
- Рекомендуемая структура именования: `{color}_{piece}.gltf`
  - `white_king.gltf`, `black_queen.gltf` и т.д.

### Где взять модели:
1. [Sketchfab](https://sketchfab.com/) - много бесплатных моделей
2. [TurboSquid](https://www.turbosquid.com/)
3. [Kenney.nl](https://kenney.nl/) - бесплатные ассеты
4. Создать самостоятельно в Blender, Maya, 3ds Max

### Как импортировать:
1. Скачайте или создайте 6 типов фигур для каждого цвета (всего 12 файлов)
2. Экспортируйте в формате glTF 2.0
3. Поместите файлы в папку `src/main/resources/models/`
4. Назовите файлы по шаблону:
   - `white_king.gltf`, `white_queen.gltf`, `white_rook.gltf`, `white_bishop.gltf`, `white_knight.gltf`, `white_pawn.gltf`
   - `black_king.gltf`, `black_queen.gltf`, `black_rook.gltf`, `black_bishop.gltf`, `black_knight.gltf`, `black_pawn.gltf`

Если модели не найдены, игра автоматически создаст простые геометрические фигуры-заглушки.

## Система управления

### Для игры на одном компьютере (2 человека):
- **Стрелки влево/вправо** - вращение камеры вокруг доски
- **Стрелки вверх/вниз** - изменение высоты камеры (зум)
- **ЛКМ (левая кнопка мыши)** - выбор фигуры / ход
- **ПКМ (правая кнопка мыши)** - отмена выбора
- **Колесо мыши** - быстрый зум

### Для игры с ботом:
То же управление, вы играете за белых, бот отвечает за черных

### UI элементы:
- Главное меню с выбором режима игры
- Панель статуса (чей ход, режим игры)
- Кнопки "Отменить ход" и "Меню"

## Режимы игры

1. **Против бота** - игра за белых против AI (алгоритм минимакс с альфа-бета отсечением)
2. **Два игрока** - локальная игра на одном компьютере по очереди

## Особенности реализации

- **3D графика** - полноценная трехмерная сцена с освещением и тенями
- **Физика** - интегрирован физический движок (для будущих улучшений)
- **AI бот** - использует алгоритм минимакс с глубиной просчета 3 полухода
- **История ходов** - возможность отмены последнего хода
- **Адаптивные материалы** - если нет моделей, используются процедурные геометрические фигуры
